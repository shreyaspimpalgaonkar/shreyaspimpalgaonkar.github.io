Let $k$ be a field, let $V$ be a $k$-vector space of dimension $d$, and let $G\subseteq GL(V)$ be a finite group. Set $r:=\dim_k (V^*)^G$ and assume $r\ge 1$. Let $R:=k[V]^G$ be the invariant ring, and write its Hilbert quasi-polynomial (for $n\gg 0$) in the form \[ H_R(n)=a_{d-1}(n)n^{d-1}+a_{d-2}(n)n^{d-2}+\cdots+a_1(n)n+a_0(n), \] where each $a_i(n)$ is a periodic function of $n$. Compute the sum of the indices $i\in\{0,1,\dots,d-1\}$ for which $a_i(n)$ is constant.

An oriented graph is a directed graph with no pair of opposite arcs. For a digraph $D$, let $fvs(D)$ be the minimum size of a vertex set whose deletion makes $D$ acyclic, and let $\Delta(D)$ be the maximum (total) degree, i.e., for each vertex $v$, $\deg(v)=\deg^+(v)+\deg^-(v)$ and $\Delta(D)=\max_v \deg(v)$. Define \[ c^*:=\inf\Bigl\{c\in\mathbb R:\ \forall\text{ oriented graphs }D\text{ on }n\text{ vertices with }\Delta(D)\le 4,\ fvs(D)\le c\,n\Bigr\}. \] What is the exact value of $c^*$?

Let $(A,\Theta)$ be a very general principally polarized complex abelian variety of dimension $6$. Let $\theta:=c_1(\Theta)\in H^2(A,\mathbb Z)$, and define the minimal curve cohomology class to be \[ \gamma:=\frac{\theta^5}{5!}\in H^{10}(A,\mathbb Z), \] which is the class Poincaré dual to the minimal (with respect to $\Theta$) integral Hodge class of a 1-cycle. Determine the smallest positive integer $m$ such that the class $m\gamma$ is represented by an algebraic $1$-cycle on $A$ (i.e. $m\gamma$ lies in the image of the cycle class map $\mathrm{CH}_1(A)\to H^{10}(A,\mathbb Z)$).

research math question answeringexact public sample

Let $p>1$. Consider the following asynchronous $\ell^p$-relaxation dynamics on a connected finite graph $G=(V,E)$ with $|V|=n$, where $V=I\sqcup B$ with $I\neq\emptyset$ (interior) and $B\neq\emptyset$ (boundary). Fix an initial profile $f_0:V\to[0,1]$ and keep boundary values fixed, i.e. $f_t(b)=f_0(b)$ for all $b\in B$ and $t\ge 0$. For each integer time $t\ge 1$, pick $v_t\in I$ uniformly at random and update only that vertex by setting $f_t(v_t)$ to be the (unique) minimizer of $x\mapsto \sum_{w\sim v_t}|x-f_{t-1}(w)|^p$, while keeping $f_t(u)=f_{t-1}(u)$ for $u\neq v_t$. Let $h:V\to\mathbb R$ be the (a.s. unique) limit of $f_t$ (the $p$-harmonic extension of $f_0|_B$), and for $\varepsilon\in(0,1/2]$ define the approximation time $\tau_p(\varepsilon):=\min\{t\ge 0:\|f_t-h\|_\infty\le\varepsilon\}$. Define $\beta_p$ to be the optimal (smallest possible) exponent such that there exist constants $C(p),c(p)>0$ with the following worst-case bound holding for all $n$, all such graphs with $n$ vertices, and all initial profiles $f_0$: for $p\in[2,\infty)$, $\mathbb E[\tau_p(\varepsilon)]\le C(p)\,n^{\beta_p}(\log(n/\varepsilon))^{c(p)}$, and for $p\in(1,2)$, $\mathbb E[\tau_p(\varepsilon)]\le C(p)\,n^{\beta_p}\varepsilon^{-c(p)}$. If $p < 3$, what is $\beta_p$ as an explicit function of $p$?
