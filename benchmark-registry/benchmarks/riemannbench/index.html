research level mathematics problemexact public sample

Find the number of distinct isomorphism classes of multibasic A-modules M over the Hahn-series valuation ring A satisfying three structural conditions on K tensor End(M), F tensor End(M), and the left-continuity exceptions of e_r.

public paper or web example

Problem: Let F be the field of order 2. Let K be the field of Hahn series in indeterminate t with value group R and residue field F. Let A be the subring of K consisting of those a in K with non-negative valuation. Consider K as an A-module. For q in R, let I_q = t^q A and I_{>q} = union_{r>q} I_r. Write A/I_{>0} as F, since they are identical both as A-modules and as fields. Let Theta = K/I_{>0} and Phi = K/A. We say that an A-module M is basic if it is isomorphic to L/N for some N < L <= K, and multibasic if it is a direct sum of a finite list of basic modules. Find the number of distinct isomorphism classes of multibasic A-modules M satisfying: (i) K tensor End(M) = K; (ii) F tensor End(M) = F; (iii) for e_r = dim_F(F tensor I_r Hom(I_{>0}, M)), lim_{p to q^-} e_p = e_q for all real q > 0 except for integers q with 29 <= q <= 328. If the answer is infinite, write -1.

research level topological recursion problemexact public sample

Calculate the genus-2 free energy F_2 for the Eynard-Orantin topological recursion formalism on the spectral curve CP^1 with x = t + 1/t, y = t^3/3, and the standard bidifferential.

public paper or web example

Problem: Consider the Eynard Orantin Topological Recursion Formalism for the spectral curve (CP^1, x, y, omega_{0,2}(x, y)), where x = t + 1/t and y = t^3 / 3, and the fundamental bidifferential is omega_{0,2}(x_1, x_2) = dz_1 dz_2 / (z_1 - z_2)^2, with z_1, z_2 in CP^1. Note that x has two simple ramification points at +/- 1 of order 2 with deck transformation theta(t) = 1/t. Please calculate the free energy F_2 and return it as a rational fraction in the format a/b for a and b coprime. Recall that F_g = 1/(2g-2) sum_{a in Delta} Res_{q=a} Phi(q) omega_{g,1}(q), where Phi(q) = integral_o^q y(t) dx(t), for an arbitrary base point o.

research level graph theory problemexact public sample

A robust maximal independent set in a graph G is a maximal independent set that remains maximal in all connected spanning subgraphs of G. The public Riemann-bench leaderboard asks how many connected graphs on 12 vertices have the property that every maximal independent set is robust, up to isomorphism.

public paper or web example
